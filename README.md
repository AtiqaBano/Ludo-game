# Ludo-game
A fully functional Ludo Board Game built with HTML5, CSS3, and Bootstrap. Features include real-time dice rolling, multi-token movement, safe zones, and killing logic with smooth animations.    ludo-game javascript-games frontend-development html5-canvas css-animations web-app


🚀 Features
Realistic Gameplay: Tokens stay in 'Home' until a 6 is rolled.

Dynamic Movement: Smooth animations as tokens navigate the board.

Combat System: 'Killing logic' enabled—land on an opponent's tile to send them back home!

Safe Zones: Designated star tiles where tokens are protected from being captured.

Responsive UI: Styled with Bootstrap and custom CSS for a polished, centered look on all screens.

Multiplayer Support: Integrated logic for 2-player and 4-player modes.

🛠️ Technologies Used
Frontend: HTML5, CSS3 (Flexbox & Grid)

Framework: Bootstrap 5

Logic: Vanilla JavaScript (ES6)

🎮 How to Play
Open index.html in your browser.

Click Start Game and select the number of players.

Click the Roll button to roll the dice.

If you roll a 6, click a token to move it onto the board.

Navigate all your tokens to the finish line to win!
